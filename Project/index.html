Let's talk movie ratings

Introduction


In the context of the W209 - Data Visualization and Communication class at the MIDS program from UC Berkeley, our group has focused on visualizing information about movie ratings. The general concept is to analyze movie ratings from different perspectives. For example, we hoped to answer questions such as: How do ratings vary by genre? How do rating patterns vary by demographics? How do they vary by geography? We hope that this presentation can reach a broad audience including curious movie fans, advertisers and media consultants, as well as studio executives.

The dataset used for this project comes from MovieLens (http://grouplens.org/datasets/movielens/). It includes slightly over 20 million ratings from 138493 users across 27278 movies. Ratings are timestamped and are on a scale from 1 to 5. The information in the ratings includes movie titles, release dates, genres, as well as user information such as age, gender, occupation and zip code. The dataset was compiled in March 2015 so information pertaining to newer movies since that time is not included. Tableau is the principal tool used to create the visualizations.

Helpful usage instructions for the dashboards are listed below:

  • All dashboards are interactive. Hovering on them or clicking elements can provide additional functionality

  • Elements such as bars or circles can be selected to highlight their information

  • Multiple elements can be selected by performing a CTRL+Click

  • Hovering over filters presents an option to clear the filter or apply an include or exclude option

  • Sliders within filters can be changed by manually typing in numbers rather than dragging the slider

  • Hovering over the map presents additional options such as search, select, zoom and pan

  • There is a reset button at the bottom of each dashboard that allows it to be re-initialized

Moviegoer Dashboards

This section presents summary level information about the ratings dataset. In particular it touches on the top movies and the distribution across genres. The visualizations here are intended for all moviegoers to get an idea of the top movies and genres. When calculating the top movies, a function of the average rating and the number of ratings is used. This is done in order to prevent movies with very few ratings from taking over the top spots.

Marketer Dashboards

This section presents ratings information broken down by different demographics. It is primarily intended for media consultants and marketers to be able to identify key demographics that they may wish to target.

Studio Exec Dashboards

This section presents overall ratings information for genres and by geographic breakdown in the USA. It is primarily intended for studio executives so that they can know who is watching which movies, and where are the viewers mostly located.
